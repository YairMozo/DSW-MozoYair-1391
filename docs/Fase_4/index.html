Responda las siguientes preguntas

1.¿Cuál es la forma correcta de especificar una media query en CSS?

a. @query (min-width: 768px) { ... }
b. @media (min-width: 768px) { ... }
c. @screen (min-width: 768px) { ... }

2.¿Qué propiedad de las media queries se utiliza para adaptar el estilo según la orientación de un dispositivo?

a. width
b. device-width
c. orientation

3. ¿Cuál es la media query correcta para apuntar a dispositivos con una resolución máxima de 1024px?

a. @media (min-resolution: 1024px) { ... }
b. @media (max-width: 1024px) { ... }
c. @media (max-resolution: 1024px) { ... }

4.¿Cuál es la sintaxis correcta para aplicar una media query solo a dispositivos móviles?

a. @media (mobile) { ... }
b. @media (max-width: 480px) { ... }
c. @media (device: mobile) { ... }

5.¿Cuál es el uso adecuado de las media queries para aplicar estilos a pantallas retina de alta resolución?

a. @media (min-resolution: 2dppx) { ... }
b. @media (min-device-pixel-ratio: 2) { ... }
c. @media (min-width: 1920px) { ... }
